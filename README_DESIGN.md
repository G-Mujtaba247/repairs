## 🚀 Quick Start

### 1. Install & Start
```bash
# Terminal 1: Website
cd ui/website && npm install && npm run dev

# Terminal 2: Admin
cd ui/admin && npm install && npm run dev

# Terminal 3: Server
cd server && npm install && npm run seed && npm run dev
```

### 2. Access
- Website: http://localhost:5173
- Admin: http://localhost:5174
- Server: http://localhost:5000
